# test-assignment-layout
Тестовое задание на позицию Junior Frontend. Адаптивная верстка, выполненная по методологии БЭМ. Включает сложную стилизацию элементов (CSS Clip-Path, Flexbox) и кроссбраузерную оптимизацию.
# Technical Assessment: Adaptive Web Layout

This repository contains the solution for a Junior Frontend Developer coding challenge.

## Project Goal
To create a fully responsive web layout based on a Figma mock-up, ensuring semantic HTML5, BEM methodology, and cross-browser compatibility.

## Key Technologies Used
* **HTML5** (Semantic structure)
* **CSS3** (Full responsiveness, CSS Flexbox)
* **Advanced Styling:** Custom geometric shapes using `clip-path` (for the button).

## Project Structure
* `index.html`: Main HTML file.
* `style.css`: All project styles (BEM structure, media queries).
* `images/`: Folder with optimized assets.
